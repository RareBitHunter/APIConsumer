# README

This project is a test programme for API consumption, and makes use of https://jsonplaceholder.typicode.com/. A free fake API for testing and prototyping.

## Description

{JSON}Placeholder is a REST API (REpresentational State Transfer), which stands for:

- **Representational** - The data you send / receive is not sent as-is. It's instead represented in another form - commonly JSON.
- **State** - The state of the resource itself, which client can change or view.
- **Transfer** - The data itself (nouns) being transferred over the network using HTTP methods (verbs.)The most common verbs being:
  - **GET -** retrieve data.
  - **POST** - create data.
  - **PUT / PATCH** - update data.
  - **DELETE** - remove data.
